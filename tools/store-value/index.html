Math

Topup Packs

Each pack has a one-time first-buy bonus. After that you get the base + standard bonus.

Rate = Origeometry / USD (higher is better)

Cost/Unit = USD / Origeometry (lower is better)

Protocol Pass

Tiers are additive: buying Customized gives you Basic + Originium + Customized rewards.

Each tier shows cumulative totals at max level (60). Items with a (+N) marker show what that specific tier adds on top of the previous.

Sanity Values

Sanity equivalents estimate how much sanity you'd spend farming the same items in protocol spaces. For each farmable item, we pick the most efficient protocol space tier:

Sanity per item = Stage cost / Drop quantity

The best (lowest) rate across all tiers is used. Emergency Sanity Boosters are valued at 40 sanity each (their direct restore amount). Cosmetics, currencies, and other non-farmable items show a dash.
